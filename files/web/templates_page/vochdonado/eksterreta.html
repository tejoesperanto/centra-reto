TEJO-Voĉo

La Reglamento pri Voĉdonadoj enkondukas du novajn voĉdonsistemojn: La Paroranga Sistemo (PR) kaj Unuopa Transdonebla Voĉo (UTV), kiuj estas tro komplikaj por efika permana kalkulado.

Por uzi tiujn ĉi voĉdonsistemojn dum ĉeestaj kunsidoj, la Reglamento permesas uzadon de komputilaj iloj okaze de sekreta, papera baloto. Por tiu ĉi celo ekzistas du programoj:

TEJO Voĉo (GUI)

TEJO Voĉo (GUI) estas eksterreta, uzantamika programo por kalkulado de balotrezultoj laŭ la Reglamento pri Voĉdonadoj. La programo haveblas por Linux, Windows kaj Mac OS.

Eluŝuti por Linux aŭ Windows

Fontkodo (GitHub, MIT-permesilo)

Por uzi la programon en Mac OS, necesas instali Python 3 kun Pip kaj tiam fari: 

pip3 install PyQt5
python3 src/main.py

TEJO Voĉo (Term)

TEJO Voĉo (Term) estas eksterreta programo por kalkulado de balotrezultoj laŭ la Reglamento pri Voĉdonadoj. La programo haveblas nur por Linux.
